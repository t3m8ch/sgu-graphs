# Практика по графам Кудякова Артёма 351 группы, факультет КНиИТ СГУ им. Чернышевского

## Как воспроизвести домашнее задание на 7 ноября?

1. Установить инструментарий для комплиции Rust-проектов: https://rust-lang.org
2. Склонировать репозиторий
3. Перейти в директорию проекта: `cd sgu-graphs`
4. Скомпилировать и запустить проект: `cargo run`
5. Загрузить файл `danilasar_homework.json`
6. Ввести команду `mst_prim 1`
7. Получаем минимальное остовное дерево!
